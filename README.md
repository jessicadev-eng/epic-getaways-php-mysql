Web-based e-commerce application developed using PHP and MySQL.

## Features
- User registration and login with session management
- Product listing and search (partial search by name)
- Shopping cart functionality
- Checkout process and order storage in database
- Server-side validation and basic security practices

## Technologies
- PHP
- MySQL
- HTML / CSS
- XAMPP (Apache, MySQL)

## How to run locally
1. Install XAMPP
2. Start Apache and MySQL
3. Copy the project folder into `xampp/htdocs`
4. Import the provided SQL file into phpMyAdmin
5. Open the browser and go to:
   http://localhost/epic-getaways/
